Simulateur d'investissements









Bien sûr InvestmentSimulator ! Voici une explication plus structurée sur l'utilisation du site :

  1. Remplir le formulaire :
    • Durée (années) : Indiquez la période d'investissement en années.
    • Salaire initial : Entrez votre salaire initial en euros.
    • Taux d'épargne (%) : Indiquez le pourcentage de votre salaire que vous souhaitez épargner.
  2. Ajouter des investissements :
    • Cliquez sur le bouton "+" à côté du champ "Nom de l'investissement" pour ajouter un nouvel investissement.
    • Pour chaque investissement ajouté, remplissez les champs suivants :
      • Nom de l'investissement : Donnez un nom à votre investissement.
      • Taux de rendement annuel (%) : Indiquez le pourcentage de rendement annuel attendu pour cet investissement.
      • Montant initial : Entrez le montant initial que vous souhaitez investir.
      • Pourcentage d'allocation (%) : Indiquez le pourcentage de votre portefeuille total que vous souhaitez allouer à cet investissement (ne doit pas excéder 100% entre tout vos investissement).
      • Frais de gestion annuels (%) : Entrez le pourcentage des frais de gestion annuels pour cet investissement.
  3. Simuler les investissements :
    • Une fois que vous avez ajouté tous vos investissements, cliquez sur le bouton "Simuler".
    • La page affichera un graphique montrant l'évolution de la valeur totale de votre portefeuille d'investissements au fil du temps.
  4. Visualiser les performances :
    • Si vous souhaitez visualiser les performances individuelles de vos investissements, cliquez sur le bouton "Visualiser".
    • Un graphique séparé s'affichera pour chaque investissement, montrant son évolution au fil du temps.

C'est tout ! Avec ces étapes, vous pourrez utiliser le simulateur d'investissements pour calculer et visualiser les performances de votre portefeuille d'investissements sur une période donnée. N'hésitez pas à ajouter plusieurs investissements pour voir comment ils se comportent ensemble dans votre portefeuille.
